# Food Delivery Platform (Learning Edition)

This repository will contain a learning-oriented full-stack food delivery platform:
- Backend: Spring Boot + MySQL
- Admin Web: lightweight management UI
- WeChat Mini Program: native mini program client

Planned modules:
- User, merchant, rider, admin roles
- Auth, product catalog, cart, order, payment (mock), delivery, reviews, coupons
- Admin management for merchants, users, orders, content

Next steps: add backend, database schema, mini program, admin web.
